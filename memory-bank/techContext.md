# Tech Context

## Domain Terms (O-ISAC)
*   **O-ISAC:** Optical Integrated Sensing and Communication.
*   **Fiber-ISAC:** Sensing (DAS, vibration) + Comms on fiber.
*   **Wireless O-ISAC:** FSO, VLC, LiDAR-based ISAC.
*   **NANF:** Nested Anti-resonant Nodeless Fiber (a type of Hollow Core Fiber).
*   **DAS:** Distributed Acoustic Sensing.
*   **OFDM:** Orthogonal Frequency-Division Multiplexing (common waveform).
*   **FMCW:** Frequency Modulated Continuous Wave (common radar waveform).
*   **IM/DD:** Intensity Modulation / Direct Detection.
